1. PANEL PRINCIPAL (Dashboard)
El centro de mando, la primera vista al entrar.

1.1. Vista General Ejecutiva

Resumen de cumplimiento global (porcentaje general).

Gráfico de cumplimiento por marco normativo (ISO 27001, GDPR, etc.).

Indicadores clave de riesgo (KRI) en formato de semáforo.

Calendario de hitos importantes (próximas auditorías, fechas límite de tareas).

1.2. Mis Tareas Pendientes

Vista filtrada exclusivamente para el usuario logueado.

Lista de tareas asignadas, con prioridad y fecha de vencimiento.

Acceso rápido para completar tareas o subir evidencias.

1.3. Alertas y Notificaciones

Centro de mensajes con alertas del sistema (controles vencidos, brechas detectadas, tareas atrasadas).

Historial de notificaciones ya leídas.

2. GESTIÓN DE NORMAS Y CONTROLES (Frameworks)
El corazón de la configuración del sistema, donde se definen las reglas del juego.

2.1. Biblioteca de Marcos Normativos

Lista de frameworks disponibles (precargados o personalizados).

Vista detallada de cada marco (dominios, categorías, objetivos de control).

2.2. Mapeo de Controles

Matriz que relaciona los controles internos de la empresa con los requisitos de los diferentes marcos.

Funcionalidad de "vista de cumplimiento múltiple" (ver qué controles sirven para GDPR y SOX a la vez).

2.3. Catálogo de Controles Internos

Inventario maestro de todos los controles que la empresa ha implementado.

Cada control con su descripción, responsable, frecuencia de revisión y estado.

3. RIESGOS (Risk Management)
El módulo para identificar, evaluar y mitigar los riesgos de cumplimiento.

3.1. Registro de Riesgos

Lista maestra de todos los riesgos identificados.

Vista con valoración (Impacto x Probabilidad) y tratamiento asignado.

3.2. Evaluación de Riesgos

Asistente para crear nuevas evaluaciones (por proyecto, por área, por activo).

Cuestionarios dinámicos para calcular el nivel de riesgo.

3.3. Plan de Mitigación

Seguimiento de las acciones para reducir los riesgos.

Enlace directo con el módulo de tareas.

4. AUDITORÍAS Y REVISIONES
El espacio para planificar, ejecutar y documentar las auditorías internas y externas.

4.1. Plan de Auditorías

Calendario y planificación de las auditorías del año.

Asignación de auditores y alcance de cada auditoría.

4.2. Gestión de Hallazgos

Registro de no conformidades, observaciones y oportunidades de mejora.

Flujo de trabajo para su aprobación y cierre (con planes de acción asociados).

4.3. Programas de Pruebas (Test Plans)

Definición de los procedimientos para probar la eficacia de los controles.

Registro de los resultados de las pruebas (exitosas/fallidas).

5. EVIDENCIAS Y DOCUMENTOS
El repositorio centralizado y seguro para toda la documentación de cumplimiento.

5.1. Repositorio de Evidencias

Visor de archivos organizado por controles, fechas o áreas.

Control de versiones (historial de cambios de un documento).

5.2. Políticas y Procedimientos

Biblioteca de documentos normativos internos.

Control de vigencia y fechas de revisión de cada política.

5.3. Subida Masiva

Herramienta para cargar múltiples evidencias a la vez y asociarlas automáticamente.

6. TAREAS Y FLUJOS DE TRABAJO (Workflows)
El motor operativo que asigna y da seguimiento a las acciones concretas.

6.1. Gestión de Tareas

Vista de todas las tareas del sistema (para administradores).

Filtros por responsable, estado, fecha, control asociado.

6.2. Diseñador de Flujos de Aprobación

Herramienta visual para crear workflows personalizados (ej: "Si se sube una política, debe ser aprobada por el Responsable de Cumplimiento y luego por el CISO").

6.3. Tareas Recurrentes

Configuración de tareas periódicas (ej: "Revisar accesos privilegiados" cada mes).

7. INFORMES (Reports)
La fábrica de reportes para diferentes audiencias.

7.1. Biblioteca de Informes Predefinidos

Informes ejecutivos (resumen para la dirección).

Informes de brechas (Gap Analysis).

Informes listos para auditoría (con toda la evidencia compilada).

7.2. Creador de Informes Personalizados

Asistente para seleccionar qué datos, gráficos y filtros incluir en un informe a medida.

7.3. Informes Programados

Configuración de entregas automáticas de informes por correo electrónico (diario, semanal, mensual).

8. ADMINISTRACIÓN
La configuración técnica y de usuarios del sistema (solo para roles con privilegios).

8.1. Gestión de Usuarios y Roles

Alta, baja y modificación de usuarios.

Asignación de roles y permisos (Responsable, Auditor, Empleado, Admin).

8.2. Configuración de la Organización

Datos de la empresa, estructura de áreas/departamentos.

8.3. Registros de Actividad (Logs de Auditoría)

Traza inalterable de todas las acciones realizadas en el sistema (quién, qué, cuándo).

8.4. Integraciones

Configuración de conexiones con otras herramientas (Active Directory, Slack, correo electrónico, etc.).
